This static website was made for PIUG, 2024

This website has 6 web pages

This website has a fixed navigation bar to navigate between its pages. The current page is highlighted and marked.

The Store page has a crude but functional searching / filtering function.

The Home page has a "merry-go-round" (carrousel) to display news banners.

The Store page displays game cards and the Home page displays videos on wrapped grids.

The support page has a contact form. The inputs are being validated before the form can be submitted.

Videos can be observed on the Home page.

Social media icons can be observed on top of the news cards. The idea behind is that they would link to the pages of these respective events on the respective platforms.

Both text and images are used to convey information.

Support (Help) button present on the fixed navigation bar.

Upvote and Downvote buttons present on the "game cards" on the Store page.

The website is responsive to different screen sizes.

Light/Dark mode toggle present on the Settings page.

Basic integration of Google Analytics.
